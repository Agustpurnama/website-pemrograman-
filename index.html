Belajar Python

Python adalah bahasa pemrograman yang mudah dipelajari dan banyak digunakan untuk aplikasi web, data science, automasi, dan banyak lagi.

Contoh Kode Python:

# Python code
print("Hello, World!")

Di atas adalah contoh kode Python sederhana yang mencetak pesan "Hello, World!" ke layar.

Contoh Fungsi Python:

# Fungsi untuk menghitung luas lingkaran
import math
def luas_lingkaran(radius):
    return math.pi * radius * radius

print(luas_lingkaran(5))

Ini adalah contoh fungsi Python yang menghitung luas lingkaran berdasarkan radius yang diberikan.

Belajar HTML & CSS

HTML adalah bahasa markah yang digunakan untuk membuat struktur halaman web. CSS digunakan untuk mempercantik tampilan halaman tersebut.

Contoh Kode HTML:





  Website Saya
  


  
Selamat Datang di Website Saya

  
Ini adalah contoh website sederhana menggunakan HTML dan CSS.

Ini adalah contoh kode HTML untuk membuat halaman sederhana dengan sebuah judul dan paragraf.

Contoh Kode CSS:

/* CSS code */
body {
  font-family: Arial, sans-serif;
  background-color: #f4f4f4;
}

h1 {
  color: #333;
  text-align: center;
}

p {
  color: #666;
  text-align: center;
}

Ini adalah contoh kode CSS yang memperindah tampilan halaman web yang dibuat di atas.
